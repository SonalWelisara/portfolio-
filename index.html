Selected Works.

Dressing Room

Dressing Room is a desktop application that redefines online clothing retail by offering a virtual try-on experience using a webcam. Built with React, this software leverages TensorFlow's PoseNet model to detect a user's body in real time. The application then accurately superimposes clothing items onto the user's live video feed, allowing them to visualize how different clothes fit and look before making a purchase. This innovative approach enhances the shopping experience by blending computer vision with e-commerce, helping users feel more confident in their purchasing decisions and reducing the uncertainty of online shopping.

Aquaguard

Aquaguard is a mobile application developed in alignment with the UN's Sustainable Development Goal 6: Clean Water and Sanitation. It serves as a community-driven platform for monitoring and discussing water quality. The app empowers users to actively participate by logging in and sharing information about local water sources, water quality, and potential hazards through a built-in forum inspired by Reddit. This collective data contributes to a comprehensive, real-time map of water sources and hazards. Aquaguard's core functionality not only raises awareness but also enables communities to take proactive steps toward ensuring access to safe drinking water, making it a powerful tool for social and environmental impact.

YourNotes

YourNotes is a mobile note-taking application built with Flutter, designed to provide a clean and intuitive user experience similar to Google Keep. The app allows users to create, edit, and organize their notes efficiently. To ensure fast and reliable performance, YourNotes utilizes the Hive database for local data storage, keeping all user data securely on the device's internal memory. This approach prioritizes user privacy and provides a seamless offline experience, making it a dependable tool for capturing ideas and staying organized on the go.

Tailwaggers

Tailwaggers is an online pet care system that provides a centralized platform for pet owners to access essential care resources and interact with veterinary experts. This web application streamlines pet care management by offering features like appointment scheduling and a comprehensive knowledge base of articles and FAQs. Built with HTML, CSS, and JavaScript for the frontend and PHP for the backend, this project demonstrates my ability to create a full-stack, user-friendly solution from the ground up, with a focus on delivering a seamless and efficient experience for pet owners.
